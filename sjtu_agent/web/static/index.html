📖 中国大学MOOC
📚 Canvas Token 可选 · 通常由 jAccount 自动配置
⚡ 填写上方 jAccount 账号密码后，运行 sjtu-agent login 可自动创建 Canvas Token，无需手动填写。

仅当自动配置失败时，才需手动获取：登录 oc.sjtu.edu.cn/profile/settings → 页面下方「已批准的集成」→「+ 新增访问令牌」→ 复制后粘贴到此处。

💬 微信 Bot（ilink）
基于 ilink 协议，无需第三方微信框架。配置步骤：
1. 在终端运行 sjtu-agent wechat-bot --login
2. 用微信扫码授权后，Token 将自动保存到 config.json
3. 刷新此页面即可看到【已配置】状态
完成扫码后刷新此页面查看状态。
✅ 微信 Bot 已配置（Token 已保存）。
若需重新扫码登录，在终端运行：sjtu-agent wechat-bot --login
✈️ Telegram Bot
如何获取 Bot Token：在 Telegram 中搜索 @BotFather， 发送 /newbot，按提示创建后复制 Token。

如何获取你的用户 ID：搜索 @userinfobot，发送任意消息即可看到你的 ID。
